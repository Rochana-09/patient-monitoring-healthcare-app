## Project Overview
This project is an Android-based healthcare application developed to monitor patient vital signs and manage patient health data in a structured manner. The application supports real-time tracking of vitals and provides alerts and reports to assist in data-driven clinical decision-making.

## Key Features
- Patient profile management (age, condition, vitals)
- Monitoring of vital signs such as blood pressure and heart rate
- Alert generation for abnormal health conditions
- Dashboard and reports for patient health insights
- Teleconsultation support for remote monitoring
- Secure local data storage using SQLite

## Data Handling & Analytics
- Structured storage of patient health data
- Retrieval and organization of vitals for reporting
- Alert logic based on predefined threshold values
- Dashboard-style presentation of health trends

## Technologies Used
- Android Studio
- Java
- SQLite
- XML
- Material UI Components

## BI & Analytics Relevance
- Demonstrates structured data modeling
- Applies reporting and dashboard concepts
- Supports data-driven decision-making in healthcare
- Reflects real-world operational analytics use cases

## Outcome
The application improves healthcare monitoring efficiency by providing quick access to patient data, alerts for critical conditions, and organized reports to support informed clinical decisions.
